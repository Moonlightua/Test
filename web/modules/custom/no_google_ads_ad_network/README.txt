Settings guide:
1) Create a field of the type `boolean` aka checkbox.
2) Go by path /admin/no-google-ads-ad-network/settings/ad-network
Settings form:
3) In the field `Content type` select content type for attaching `No google adds ad script`.
4) In the field `Control field` select the field which controls, the behavior of the script.
5) Save the form.

Notes:
1) Script don't attach on the admin pages, because search engines and ad networks do not have access to them.
